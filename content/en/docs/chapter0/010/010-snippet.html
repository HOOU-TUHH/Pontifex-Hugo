Operations on sets

We remember the important operations for sets:

  • \(M_1 \cup M_2 := \{ x: x\in M_1 \vee x \in M_2 \}\) (union)

  • \(M_1 \cap M_2 := \{ x : x \in M_1 \wedge x \in M_2\}\) (intersection)

  • \(M_1 \setminus M_2 := \{ x : x \in M_1 \wedge x \not\in M_2 \}\) (set difference)

Set compositions

The \(M_1\cup M_2\) is the new set that consists exactly of the objects that are elements of \(M_1\) or \(M_2\).

 

The \(M_1\cap M_2\) is the new set whose elements are the objects that are elements of \(M_1\) and \(M_2\).

 

We write \(M_1\setminus M_2\) for the whose elements are the objects that are elements of \(M_1\) but not elements of \(M_2\).

 

A of \(M_2\) is each set whose elements are also elements of \(M_2\).

Complement set

Let \(X\) be a set. Then for a subset \(M \subset X\) there is a unique of \(M\) with respect to \(X\): \[M^c := X \setminus M = \{ x \in X \st x \notin M \}\]

Product set The of two sets \(A,B\) is given as the set of all (two elements with order): \[A \times B := \{ (a,b) : a \in A, b \in B\}\]
